# Karaoke Plugin

**Plugin Key:** `karaoke`

Adds karaoke session management and song request tracking to events.

## Quick Start

```bash
just plugin-enable karaoke
just devModeFixtures
```

**Namespace:** `Plugin\Karaoke\*` — see [core guidelines](../../.claude/CLAUDE.md) and [plugin architecture](../../.claude/architecture.md#plugin-architecture).
